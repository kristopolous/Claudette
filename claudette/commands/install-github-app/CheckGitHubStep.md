## Purpose
Step component that checks GitHub CLI installation during GitHub App setup.

## Imports

- **Internal**: `Text`, `Box`

## Logic
Simple component that displays "Checking GitHub CLI installation…" while the main flow performs the actual checks (gh --version, gh auth status). This is a placeholder UI during the initial check.
