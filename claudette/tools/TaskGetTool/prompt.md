## Purpose
Provides the description and prompt text for the TaskGet tool that retrieves task details by ID.

## Imports
- **Stdlib**: None
- **External**: None
- **Internal**: None

## Logic
Defines static strings explaining when to use the tool (before starting work, understanding dependencies, after assignment), what output it returns (subject, description, status, blocks, blockedBy), and tips for effective usage such as checking blockedBy before beginning work.

## Exports
- `DESCRIPTION` - short description of retrieving a task by ID
- `PROMPT` - detailed prompt with usage guidance, output format, and tips
