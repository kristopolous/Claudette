## Purpose
Displays a simple notification that a tool use was rejected.

## Imports
- **External**: REACT
- **Internal**: `ink`, `components/MessageResponse`

## Logic
Renders a single-line MessageResponse containing a dimmed text label stating that the tool use was rejected, with no conditional logic or dynamic content.

## Exports
- `RejectedToolUseMessage` - renders a fixed rejection notification for a rejected tool use
