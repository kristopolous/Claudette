## Purpose
Renders a notification that a tool use was canceled by the user.

## Imports
- **External**: REACT
- **Internal**: `components/InterruptedByUser`, `components/MessageResponse`

## Logic
Renders a single-line MessageResponse containing an InterruptedByUser component with no additional logic or conditional branches.

## Exports
- `UserToolCanceledMessage` - renders a fixed cancellation notification for a user-interrupted tool use
