## Purpose
Renders a dimmed interruption indicator message when the user interrupts execution.

## Imports
- **Stdlib**: none
- **External**: REACT (React)
- **Internal**: ../ink (Text)

## Logic
Displays a static "Interrupted" message followed by a prompt asking what Claudette should do instead, rendered in dim color.
