## Purpose
Provides past-tense verbs used in turn completion messages to describe what the agent accomplished during a processing turn.

## Imports
- **Stdlib**: none
- **External**: none
- **Internal**: none

## Logic
Exports a small array of past-tense verbs suitable for duration-based completion messages (e.g., "Worked for 5s").

## Exports
- `TURN_COMPLETION_VERBS` - array of past-tense verbs for turn completion messages
